# HowToUseVim🏄

没有特别说明，则是在正常模式(Normal Mode)下的命令！

## 1. Exiting 退出

| 命令   | 描述                           |
| ------ | ------------------------------ |
| :q     | 关闭文件                       |
| :qa    | 关闭所有文件                   |
| :w     | 保存                           |
| :wq/:x | 保存并关闭文件                 |
| ZZ     | 保存并退出                     |
| ZQ     | 不保存并退出                   |

## 2. Exiting insert mode 离开插入模式

| 命令       | 描述                                 |
| ---------- | ------------------------------------ |
| Esc / <C - [ >| 退出插入模式                         |
| <C - C>      | 退出插入模式，并取消当前命令           |

## 3. Editing 编辑

| 命令    | 描述                          |
| ------- | ----------------------------- |
| a       | 追加                          |
| A       | 从行尾追加                    |
| i       | 插入                          |
| o       | 下一行                        |
| O       | 上一行                        |
| s       | 删除字符并插入                |
| S       | 删除行并插入                  |
| C       | 删除至行尾并插入              |
| r       | 替换一个字符                  |
| R       | 进入替换模式                  |
| u       | 撤销更改                      |
| <C-R>   | 重做更改                      |
